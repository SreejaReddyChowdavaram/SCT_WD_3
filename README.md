This project is developed as **Task 3 of SkillCraft Technology Internship Program**.

## 🚀 Features

* 👥 Player vs Player mode
* 🤖 Player vs Computer (Easy)
* 🔥 Player vs Computer (Hard)
* 🎨 Modern and responsive UI
* 🔄 Restart game option

## 🎯 Game Rules

* The game is played on a 3×3 grid
* Player 1 uses **X**, Player 2/Computer uses **O**
* First to align 3 marks (row, column, or diagonal) wins
* If all cells are filled → it's a draw
